|Librerías Empleadas en el Script|
----------------------------------
webdriver: Control del navegador
Keys: Entrada de teclado
By: Estrategias de localización
WebDriverWait: Esperas explícitas
expected_conditions (EC): Condiciones de aparición/clic
Select: Manejo de listas desplegables
Options: Configuración avanzada del navegador
time: Pausas temporales mínimas

---------------------------
|Flujo funcional del robot|
---------------------------
1. Abrir navegador Chrome en modo incógnito
2. Ingresar a la URL del reto
3. Llenar los campos: Nombre, Apellido Teléfono, País, Email, Contraseña
4. Hacer scroll automáticamente hasta el email
5. Clic en el botón Register
6. Hacer scroll hasta la sección de resultados
7. Cerrar el navegador
